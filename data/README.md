# Data Layer

This folder contains **mock source data** representing enterprise systems used for analytics and reporting.

## Source Systems

### SAP Finance
- File: `mock_sap_finance.csv`
- Represents financial postings from SAP
- Used as a primary financial data source

### Operational Systems
- File: `mock_operations.csv`
- Represents operational process data
- Complements SAP data for end-to-end analysis

## Notes
- Data is anonymized and simplified
- Structure mirrors real enterprise schemas
- Used for ingestion into BigQuery staging tables
