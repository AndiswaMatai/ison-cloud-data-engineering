# Analytics Layer

This folder contains **business-facing SQL queries** built on the unified fact table.

## Available Queries

### Cost Efficiency
- File: `cost_efficiency.sql`
- Analyzes total spend and averages by department
- Supports cost optimization initiatives

### Performance Metrics
- File: `performance_metrics.sql`
- Tracks operational status distribution
- Used for performance monitoring and reporting

## Consumption
Outputs from this layer feed:
- Executive dashboards
- Monthly performance reports
- Cost and efficiency reviews
